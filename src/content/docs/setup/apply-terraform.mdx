## Step 2 — Apply Terraform (base infrastructure)

```bash
terraform init
terraform plan
terraform apply
```

Expect roughly 20 resources (<Tooltip term="aws" /> VPC/network/compute/endpoints + <Tooltip term="gcp" /> network/compute + <Tooltip term="dx-gateway" label="DX Gateway" />).

Save the outputs — you will reference them in later steps:

```bash
terraform output
```
