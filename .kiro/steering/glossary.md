# Glossary Conventions

- Keep glossary entries in `src/data/glossary.ts` in **alphabetical order**.
- Each entry has a `term` and `definition` field.
- Definitions should be concise (one sentence max).
- Use the `<Tooltip>` component in MDX pages to reference glossary terms.
